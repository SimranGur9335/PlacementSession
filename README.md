## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/SimranGur9335/PlacementSession.git
```

### Navigate to Repository

```bash
cd PlacementSession
```

### Compile Java Program

```bash
javac FileName.java
```

### Run Java Program

```bash
java FileName
```

### Run Spring Boot Application

```bash
cd SpringBoot
mvn spring-boot:run
```

### Build Spring Boot Project

```bash
mvn clean install
```
